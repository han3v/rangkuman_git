==== Branch GIT ====
branch dalam git digunakan untuk membuat cabang dari repository utama yang biasanya disebut master atau main
Bisa digunakan untuk memanajemen pengembangan. Misal test branch, unstable branch, 
stable branch

command dari branch :
1.Git branch: melihat semua branch (local)
2. Git branch --all: melihat semua branch (public)
3. Git branch nama_branch: membuat branch baru
4. Git checkout nama_branch: untuk berpindah branch
5. Git branch -d nama_branch: untuk menghapus branch

Conflict adalah Kondisi yang terjadi ketika merging 2 branch yang memiliki perubahan pada file yang sama
